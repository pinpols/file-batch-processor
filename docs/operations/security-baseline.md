# Security Baseline
> 中文名：安全基线说明

## 1. 密钥与凭据
- 禁止在代码仓库明文保存账号密码、Token。
- 使用环境变量或密钥管理平台注入（如 Vault/KMS）。
- 默认值不允许弱口令。

## 2. 访问控制
- 数据库账号遵循最小权限：仅授予必要读写权限。
- 管理接口按网络边界限制访问。
- 生产环境严格区分人机账号。

## 3. 依赖与供应链
- 每次 PR 执行依赖漏洞扫描（OWASP Dependency-Check、Trivy）。
- 对高危漏洞（CVSS >= 9）阻断发布。
- 定期升级过期依赖并记录变更。

## 4. 传输与存储
- 上下游通信优先 HTTPS/SFTP，禁止明文 FTP（除隔离内网临时场景）。
- 敏感数据脱敏日志输出，不记录完整凭据。
- 数据备份文件需加密存储并设置保留周期。

## 5. 审计与告警
- 关键操作需可审计（发布、重放、重跑、配置变更）。
- 安全告警（认证失败、异常访问、漏洞扫描失败）必须进入值班通道。

## 6. 基线校验清单
1. 生产配置中无默认密码
2. 依赖扫描无高危漏洞
3. 数据库最小权限已验证
4. 告警通知链路联通
